## Install Instructions
### CMake
Builds the shared library and the test binary.
```
cmake -B build
cmake --build build
```
## Features
### Edit and View Paths
Simply use the GUI to create and edit paths. They will be displayed on the screen.
### Panning
Use the arrow keys to move around.
